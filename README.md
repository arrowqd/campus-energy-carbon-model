# 校园图书馆用能碳排放测算与节能改造收益评估

## 1. 项目简介

本项目对应 K 题：校园用能碳排放测算与节能改造收益评估。

本项目选择校园图书馆作为研究对象，围绕照明设备、空调设备、插座设备和公共设备等主要用电场景，建立一套“用电量 - 电费成本 - 碳排放 - 节能收益”的测算流程。

本项目的目标是通过数据整理、公式计算和方案对比，分析校园建筑当前用能情况，并比较不同节能改造方案的节电量、减排量、节省电费和投资回收期，为校园节能改造提供参考。

## 2. 研究背景

在“双碳”目标背景下，节能降碳已经成为高校校园建设和管理中的重要方向。校园建筑数量较多，用电设备集中，长期运行会产生较大的用电成本和碳排放。

图书馆作为校园中开放时间较长、人员流动较多、照明和空调使用频繁的公共建筑，具有典型代表性。因此，本项目选择校园图书馆作为研究对象，开展用能碳排放测算与节能改造收益评估。

## 3. 研究对象

本项目以校园图书馆为例，主要分析以下四类用电设备：

1. 照明设备；
2. 空调设备；
3. 插座设备；
4. 公共设备。

其中，照明设备和空调设备通常是图书馆用电的重要组成部分，也是节能改造的重点对象。

## 4. 模型思路

本项目采用以下分析流程：

```text
设备数据收集
    ↓
年用电量测算
    ↓
年电费成本计算
    ↓
年碳排放量计算
    ↓
节能改造方案设计
    ↓
节电量、减排量、节省电费和投资回收期计算
    ↓
提出节能改造建议
```

整体思路可以概括为：

```text
用电量 → 电费成本 → 碳排放量 → 节能收益 → 改造建议
```

## 5. 数据说明

由于当前暂未取得学校真实电表数据，本项目先采用模拟数据完成模型验证。

模拟数据主要依据以下参数进行估算：

- 设备数量；
- 单台设备功率；
- 每日运行时间；
- 年运行天数；
- 负载系数；
- 电价；
- 电力碳排放因子。

后续如果能够获得学校真实用电记录，可以直接替换当前模拟数据，使模型结果更加准确。

|---|---:|---:|---:|---:|---:|

## 7. 核心计算公式

### 7.1 年用电量计算公式

```text
年用电量 = 数量 × 单台功率 × 每日运行小时 × 年运行天数 × 负载系数 ÷ 1000
```

其中，功率单位为 W，计算结果单位为 kWh。

### 7.2 年电费计算公式

```text
年电费 = 年用电量 × 电价
```

本项目中可先假设：

```text
电价 = 0.8 元/kWh
```

### 7.3 年碳排放量计算公式

```text
年碳排放量 = 年用电量 × 碳排放因子
```

本项目中可先假设：

```text
碳排放因子 = 0.57 kgCO2/kWh
```

如果需要将单位换算为吨：

```text
年碳排放量/tCO2 = 年用电量 × 碳排放因子 ÷ 1000
```

### 7.4 年节省电费计算公式

```text
年节省电费 = 年节电量 × 电价
```

### 7.5 投资回收期计算公式

```text
投资回收期 = 投资成本 ÷ 年节省电费
```

投资回收期越短，说明该方案经济性越好。

## 8. 节能改造方案设计

本项目设置三类节能改造方案：

| 方案 | 改造内容 | 假设节能率 | 投资成本 | 说明 |
|---|---|---:|---:|---|
| 方案一 | LED 照明替换 | 40% | 30000 元 | 将传统照明设备替换为 LED 节能灯具 |
| 方案二 | 空调温度管理 | 10% | 5000 元 | 通过合理设置空调温度减少空调用电 |
| 方案三 | 屋顶光伏接入 | 20% | 120000 元 | 利用图书馆屋顶建设光伏发电系统 |

## 9. 方案评价指标

为了比较不同节能改造方案，本项目选取以下指标：

1. 年节电量；
2. 年减排量；
3. 年节省电费；
4. 投资回收期。

这些指标可以用于判断不同方案的节能效果、减排效果和经济收益。

## 10. 结果分析思路

通过基础数据测算，可以得到图书馆不同设备类型的年用电量，并进一步计算总用电量、年电费和年碳排放量。

通过节能方案测算，可以得到 LED 照明替换、空调温度管理和屋顶光伏接入三种方案的节电量、减排量、年节省电费和投资回收期。

从分析思路上看：

- 空调温度管理投入成本较低，适合短期优先实施；
- LED 照明替换节能效果稳定，适合中期推进；
- 屋顶光伏接入减排潜力较大，但初期投资较高，适合作为长期建设方案。

## 11. 项目结论

本项目完成了一套校园图书馆用能碳排放测算与节能改造收益评估流程。

通过该模型，可以对校园建筑的用电量、电费成本和碳排放量进行初步测算，并进一步比较不同节能改造方案的综合收益。

本项目的价值在于，它不仅能够计算当前建筑的用能和碳排放情况，还能够为校园节能改造提供决策参考。

## 12. 改造建议

根据模型分析，本项目提出以下建议：

1. 短期内优先实施空调温度管理，因为该方案投入成本较低，见效较快；
2. 中期推进 LED 照明替换，降低照明系统长期用电量；
3. 长期考虑屋顶光伏接入，提高校园可再生能源利用比例；
4. 后续应尽量接入学校真实电表数据，提高测算结果的准确性。
